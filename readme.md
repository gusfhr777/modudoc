# Modudoc Project

The Modudoc Project (모두문서 프로젝트) is a real-time collaborative document editor. It was developed as a term project for the Object-Oriented Programming course at Korea Aerospace University (KAU). Modudoc is implemented using multithreading, object-oriented programming principles, JavaFX, and a client–server architecture.
